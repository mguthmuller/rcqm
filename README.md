# Ruby Code Quality Metrics

## Usage:

rcqm [options] 

## Metrics name:
- coverage 
- coding_style 
- statistics
- tags
- complexity 
- all

## Functionalities:

### Tags (FIXME, TODO, ...) tracking
- With '-m tags' option
- You can specify other tags with '-t' option

### Not implemented yet ...
- Code Coverage
- Coding Style
- Statistics (LOC, Comments, ...)
- Cyclomatic complexity
- Dead Code
- Duplication
